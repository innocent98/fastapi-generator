COMMON WORKFLOWS
â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•

Add a New Endpoint:
  1. Create app/api/v1/endpoints/users.py
  2. Add router to app/api/v1/api.py
  3. Run and test

Add a Database Model:
  1. Create app/db/models/user.py
  2. Run: alembic revision --autogenerate -m "add user model"
  3. Run: alembic upgrade head

Add Tests:
  1. Create tests/api/test_users.py
  2. Run: make test

Docker Setup:
  1. Edit .env with your settings
  2. Run: docker-compose up -d
  3. Visit: http://localhost:8000
